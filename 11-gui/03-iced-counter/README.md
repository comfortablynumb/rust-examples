# Iced Counter

A simple counter app using iced, an Elm-inspired GUI library.

## Concepts

- Sandbox pattern (for simple apps)
- Message passing
- Declarative UI
- State management

## Key Points

**iced uses The Elm Architecture:**
1. **Model**: Application state
2. **Update**: Handle messages and update state
3. **View**: Render UI from state

## Running

```bash
cargo run
```
